Agregar VM Base
Paso 1 — Registrar maquina virtual

La VM debe estar previamente registrada en VirtualBox. Solo se registra como referencia; no se modifica.

Configurar Llaves SSH Root
Paso 2a — Autenticacion por clave publica

Se genera un par de llaves RSA de 1024 bits. La VM se iniciara temporalmente para desplegar la llave publica al usuario root via SSH. Este proceso tarda entre 30 y 60 segundos.

Crear VM de Usuario
Paso 3a — Nueva maquina desde disco compartido

Se creara una nueva VM que comparte el disco multiattach de la base. Cada VM tiene su propia capa de escritura aislada (copy-on-write).

Crear Usuario en VM
Paso 3b — Cuenta de sistema con acceso SSH

Se creara la cuenta en el sistema operativo de la VM y se desplegaran llaves SSH RSA para autenticacion sin contrasena. La VM se iniciara automaticamente durante el proceso.
